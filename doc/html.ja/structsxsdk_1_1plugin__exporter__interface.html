説明

エクスポートに関する（エクスポートの最中にのみ有効な）各種情報を保持する。

exporter_interface から派生したエクスポータプラグインにおいて、エクスポート処理に使用するシーンの情報やストリームを取得する際に使用する。

exporter_interface::do_export() の引数として渡される。

以下の関数でこのクラスを直接取得する。これ以外に関数の引数として渡される場合がある。

rendering_context_interface::get_plugin_exporter_interface()

親の方向に辿りながら形状が選択されているかどうかを調べてその結果を返す。リンク形状内のツリーを辿っている場合にも選択状態を正しく取得することができる。

trueが返された場合はget_stream_interface() で取得したストリームに、falseが返された場合はget_text_stream_interface() で取得したストリームに書き込む。
