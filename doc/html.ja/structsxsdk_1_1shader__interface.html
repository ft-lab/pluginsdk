シェーダプラグインの基本クラス

shader_interface から派生させたクラスを定義することによって、シェーダプラグインを実装する。

シェーダプラグインは表面材質のマッピングパターンに追加される。

virtual int sxsdk::shader_interface::get_default_map_mode ( void *  aux = 0) [inline, virtual]

オーバライド可。デフォルトの投影方法の値を返す。

needs_projection() でtrueを返し投影方法を選択できる状態の時に有効。

投影方法は0=X方向、1=Y方向、2=Z方向、3=ラップマップ、4=すべて=ラップマップ、5=円筒、6=球。デフォルトでは3（ラップマップ）。

オーバライド可。接線情報を必要とするかどうかを返す。

trueを返すと、マッピング設定でイメージを割り当てることができるようになり、sample_bump() 関数が有効となる。
