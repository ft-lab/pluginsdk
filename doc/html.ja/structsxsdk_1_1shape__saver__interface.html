説明

アンドゥのための、形状の幾何情報を保持するオブジェクト

ユーザがUndoメニューアイテムを選択した場合に実行される一個のアンドゥ処理は、登録された一個または複数のアンドゥアクションから構成される。

shape_saver_interfaceオブジェクトを生成すると、その時点での元の形状の幾何情報が保持される。

アンドゥを機能させるためには、形状を変更後の状態から変更前の状態に戻す（およびその逆を行う）動作をアンドゥアクションとして登録する必要がある。まずshape_saverオブジェクトに幾何情報を保持した後形状自体を変更し、set_undo_action()を呼ぶことによってアンドゥアクションを登録する。

shape_saver_interfaceの使用例として、SDKサンプルプロジェクトのsimple_undoターゲットを参照。

形状の種類によって、以下の幾何属性が保持される。保持される幾何属性は将来版Shadeで増える可能性がある。また、旧版Shadeでは、ここで記述した属性が全てサポートされていない可能性もある。

ポリゴンメッシュ形状 (polygon_mesh_class)

頂点座標、面、稜線、丸め、スムーズ閾値、スムーズフラグ、稜線を表示フラグ

線形状 (line_class)

線形状を成すコントロールポイント

パート (part_class)

（パートが自由曲面の場合に）自由曲面を成すコントロールポイント

光源形状 (light_class)

明るさ、位置、向き、角度、ソフトネス、シャドウ、ソフトシャドウ、鏡面反射、拡散反射、減衰

球 (sphere_class)

球を定義する変換マトリクス

円 (disk_class)

円を定義する変換マトリクス
