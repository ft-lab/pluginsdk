説明

ボーンにアクセスするためのインターフェース。

ボーンに関する設定値は形状の属性の一部として保持されているが、プラグイン／スクリプトからのアクセスを容易にするため、別途インターフェイスを介して値の取得／設定を行う。

以下の関数でこのクラスを直接取得する。これ以外に関数の引数として渡される場合がある。
