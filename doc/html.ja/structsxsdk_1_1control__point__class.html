説明

コントロールポイントを表す。線形状のコントロールポイントの操作に使用する。

線形状は０個以上のコントロールポイントから成るが、control_point_classはその内の一個のコントロールポイントを表す。

以下の関数でこのクラスを直接取得する。これ以外に関数の引数として渡される場合がある。

sxsdk::shape_class::control_point()
