virtual int sxsdk::ik_class::has_ik ( sxsdk::shape_class &  shape,
bool  depth,
void *  aux = 0 
) [virtual]

指定の形状でIK情報を持つか調べる。

戻り値が0の場合はIK情報なし。
1 : IKルートを持つ
2 : IKエンドを持つ
4 : IKゴールを持つ
IKルートとIKエンドを共有する場合は 1 + 2 = 3が返される。
