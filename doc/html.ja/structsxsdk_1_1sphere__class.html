virtual sxsdk::mat4 sxsdk::sphere_class::get_matrix ( void *  = nullptr) const [virtual]

球を定義する変換マトリクスを取得する。

単位球（原点に位置し半径が1、Y軸が極軸の球）から対象とする球に変換する4x4座標変換マトリクス。unmatrix関数でこのマトリクスをスケール、回転、移動の成分に分解することができ、この場合、それぞれの成分は球の半径、向き、位置ということになる。
