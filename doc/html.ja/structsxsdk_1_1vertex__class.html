説明

ポリゴンメッシュの頂点を表すクラス

ポリゴンメッシュ形状は０個以上の頂点を持つが、vertex_classはその内の一個の頂点を表す。

以下の関数でこのクラスを直接取得する。これ以外に関数の引数として渡される場合がある。

polygon_mesh_class::vertex()
