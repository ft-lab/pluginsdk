プラグインID

■UUID

Shade 7では個々のプラグインは32ビット整数のIDで識別されていたが、Shade 8以降では128ビットのUUIDを使用する。したがって、特に理由がない限り、プラグインデベロッパにIDの範囲を割り当てたり、管理したりする必要はない。プラグインデベロッパは必要に応じて、uuidgenコマンド等でIDを生成する。

従来の32ビット整数IDを使ったプラグインも、Shade 8以降では内部的に128ビットUUIDに変換するため、無変更で動作する。

■UUID生成方法

Windows XP / Vista / 7 + Visual C++ 2008/2010

  • Visual Studio 2010 のツールメニューから「GUIDの作成」
  • 生成されたUUIDを「Copy」ボタンでクリップボードにコピー
  • コピーされたUUIDを下記の形式に編集
 // {8103127E-6E91-467f-8B98-9E6821DCC7CE}
 const uuid_class SAMPLE_UUID = uuid_class("8103127E-6E91-467f-8B98-9E6821DCC7CE");

Mac OS X

  • ターミナルで「uuidgen」を実行
  • 生成されたUUIDを下記の形式に編集
 const uuid_class SAMPLE_UUID = uuid_class("8103127E-6E91-467f-8B98-9E6821DCC7CE");
