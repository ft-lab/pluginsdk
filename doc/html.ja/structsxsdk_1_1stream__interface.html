説明

属性ストリームやファイルの読み込み、書き込みに使用します。

属性ストリームはプラグインで使用するデータの記憶領域で、形状を始めとしたShadeのオブジェクトに付随させることができる。アクセスにはstream_interface を用い、ファイルへの読み書きと同じように使用できる。

以下の関数でこのクラスを直接取得する。これ以外に関数の引数として渡される場合がある。

surface_interface, surface_class, sxsdk::shape_class, shader_node_class, scene_interface, rendering_interface, preference_interface, mapping_layer_class, distant_light_item_class, camera_class の create_attribute_stream_interface_with_uuid(), get_attribute_stream_interface_with_uuid()
