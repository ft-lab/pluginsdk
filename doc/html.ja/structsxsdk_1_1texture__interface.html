説明

texture_interfaceを介して、プラグインからShade内部のシェーディング計算ルーチンをプラグインから利用できる。

texture_interfaceは各種内部ステートを保持し、そのステートを元にシェーディング計算を行って結果を返す。

プラグインからは、texture_interfaceを生成した後、「各種ステートを設定-update()呼び出し-各種結果を取得」という処理を繰り返すことになる。
