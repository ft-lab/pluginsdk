virtual bool sxsdk::part_class::is_master_shape ( void *  = nullptr) const [virtual]

パートがマスターオブジェクトパートか判定する。

マスターオブジェクトパートマスターサーフェスパートと同様に、ルートパートの直下に１つだけ存在する。

このパート内の形状はレンダリング対象とならないので、リンク形状の親を格納してライブラリ化できる。また、全ての外部参照形状はこのパートの中に配置される。
