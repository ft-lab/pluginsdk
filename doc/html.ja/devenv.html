開発環境

■プラグインの開発環境

Shade Plugin SDKによるプラグイン開発言語はC++です。プラグインを開発するには以下に示す開発環境が別途必要になります。

Windows版

OS

  • Windows 7 32bit / 64bit
  • Windows Vista 32bit / 64bit
  • Windows XP Professional x64 Edition
  • Windows XP Professional / Home Edition / Media Center Edition (SP2以降)

開発ツール

  • Visual Studio 2010 Ultimate / Premium / Professional Edition
  • Visual C++ 2010 Express Edition

※ Windows 95 / Me / NT / 2000 には対応していません。
※ Visual Studio .NET 2003、Visual C++ .NET 2003、Visual Studio .NET 2002、Visual C++ .NET 2002、Visual C++ 6.0には対応していません。
※ Visual Studio 2010の利用言語は、Visual C++ 2010となります。
※ Visual C++ 2010 Express Edition単体では、64bit版のプラグインは開発できません。
※ Visual Studio 2010およびVisual C++ 2010利用時は、Visual Studio 2010 Service Pack 1の適用を推奨します。

　 Visual Studio 2010 Service Pack 1
　 http://www.microsoft.com/ja-jp/download/details.aspx?id=23691

※ Visual C++ 2010 Express Editionでの開発については ■Visual C++ 2010 Express Editionでのプラグイン開発について を参照してください。

もしくは

  • Visual Studio 2008 Team System / Professional Edition / Standard Edition
  • Visual C++ 2008 Express Edition

※ Windows 95 / Me / NT / 2000 には対応していません。
※ Visual Studio .NET 2003、Visual C++ .NET 2003、Visual Studio .NET 2002、Visual C++ .NET 2002、Visual C++ 6.0には対応していません。
※ Visual Studio 2008の利用言語は、Visual C++ 2008となります。
※ Visual C++ 2008 Express Editionでは、64bit版のプラグインは開発できません。
※ Visual Studio 2008およびVisual C++ 2008利用時は、Visual Studio 2008 Service Pack 1の適用を推奨します。

　 Visual Studio 2008 Service Pack 1
　 http://www.microsoft.com/japan/msdn/vstudio/downloads/sp/vs2008/sp1/

※ Visual C++ 2008 Express Editionでの開発については ■Visual C++ 2008 Express Editionでのプラグイン開発について を参照してください。

Mac OS X版

OS

  • Mac OS X 10.6 以降

開発ツール

  • Xcode 3.2 以降 / Xcode 4.3 / Xcode 4.4

※ Classic環境あるいはMac OS 9、MacOSX10.5以前には対応しておりません。
※ CodeWarrior、Project Bulider、Xcode3.2以前には対応しておりません。
※ Xcode 4.4にてプラグインを作成する際はプロジェクトの設定を変更する必要があります。詳細は ■Xcode 4.4でのプラグイン開発について を参照してください。

　 Apple Developer Connection
　 http://developer.apple.com/jp/

■Visual C++ 2010 Express Editionでのプラグイン開発について

Visual Studio 2010 ではExpress / Professional Edition以上にて、Platform SDKは組み込まれていますのでPlatform SDKについての追加設定は不要です。

・ Visual C++ 2010 Express EditionおよびMicrosoft Platform SDKを以下のWebページからプログラムをダウンロードし、インストールします。

　 Visual C++ 2010 Express Edition
　 http://www.microsoft.com/japan/msdn/vstudio/express/visualc/

設定が完了するとShade 13 プラグインの開発が行えるようになります。

■Visual C++ 2008 Express Editionでのプラグイン開発について

Visual C++ 2008 Express EditionでShade 13 プラグインの開発を行うには、Microsoft Platform SDKが必要です。
Visual Studio 2008 Standard Edition以上では、Platform SDKは組み込まれていますので設定は不要です。

・ Visual C++ 2008 Express EditionおよびMicrosoft Platform SDKを以下のWebページの下の「過去のバージョン」ページからプログラムをダウンロードし、インストールします。

　 Visual C++ 2008 Express Edition
　 http://www.microsoft.com/japan/msdn/vstudio/express/visualc/

　 Microsoft Platform SDK (「PSDK-x86.exe」をダウンロード)
　 http://www.microsoft.com/downloads/details.aspx?FamilyId=A55B6B43-E24F-4EA3-A93E-40C0EC4F68E5

・ インストールが完了したら下記ページの説明に従ってVisual C++ 2008 Express Editionの設定を行ってください。

　 Visual C++ 2008 Express Edition と Microsoft Platform SDK を一緒に使う
　 http://www.microsoft.com/japan/msdn/vstudio/express/visualc/usingpsdk/

設定が完了するとShade 13 プラグインの開発が行えるようになります。

■Xcode 4.4でのプラグイン開発について

Mac OS X 10.8 (Mountain Lion)環境のXcode 4.4にてプラグインを作成する際は、サンプルプロジェクトの設定を変更する必要があります。

pluginsプロジェクトを選択し、Build Settingsでの ArchitecturesグループにてBase SDKの選択を「OS X 10.7」とします。
これで、OS X 10.7/10.8にて動作するプラグインのビルドができるようになります。
ただし、Xcode 4.4でビルドしたプラグインはOS X 10.6以前では動作しません。
OS X 10.6でも動作するプラグインをビルドするには、OS X 10.7上のXcode 4.3を使用する必要があります。

■プラグインの動作環境

Shade 13 Plugin SDKで作成したプラグインはShade 13 Professional/Standardで動作します。Basicではプラグインがサポートされていないため動作しません。
