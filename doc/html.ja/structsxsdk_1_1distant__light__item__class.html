説明

一つの無限遠光源アイテムを表すクラス。無限遠光源の光源ごとの情報の取得、設定に使用する。

無限遠光源の設定は１個以上の無限遠光源アイテムから構成され、distant_light_item_classはその一個の無限遠光源アイテムを表す

以下の関数でこのクラスを直接取得する。これ以外に関数の引数として渡される場合がある。
