virtual void sxsdk::polygon_mesh_class::subdivide ( int  level,
void *  aux = 0 
) [virtual]

角の丸めを設定したポリゴンメッシュに対して、サブディビジョンを行う。

サブディビジョンを行うと、角の丸めで分割された頂点、面、稜線の情報を算出して内部的に保持する。

保持された頂点、面、稜線の情報は通常通りの方法で取得できる（角の丸めの適用前の情報と入れ替わる）。

情報の保持のみを行うので、形状自体は変化しない。

情報の取得後は、unsubdivide() でサブディビジョンを取り消す。

引数:
level分割レベル。0:分割しない（または最低限の分割）、1:荒い分割、2:普通の分割、3:細かい分割....
