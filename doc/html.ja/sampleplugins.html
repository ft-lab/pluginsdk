サンプルプラグイン　プロジェクト

■multi_plugin : マルチプラグイン　ターゲット

２つ（以上）のプラグインコンポーネントを一個のプラグイン（つまりWindowsではDLLファイル、Macではバンドル）に含ませる方法の例。

■simple_attribute : 属性プラグイン　ターゲット

カスタム属性を様々なオブジェクトに付加し、それを読み書き、使用する方法の例。

■simple_browser : ブラウザプラグイン　ターゲット

ブラウザポップアップメニューから起動するプラグインの例。

■simple_creator : クリエータプラグイン　サンプル

作成メニューから起動され、ユーザの図形ウインドウ内でのクリック／ドラッグに反応するプラグインの例。

■simple_dialog : ダイアログプラグイン　ターゲット

ダイアログボックスを作成／表示する方法の例。

■simple_effector : エフェクタプラグイン　ターゲット

レンダリングされたイメージに後処理を加えるプラグインの例。

■simple_exporter : エクスポータプラグイン　ターゲット

イメージのエクスポータと形状のエクスポータの例。

■simple_figure : 図形ウインドウプラグイン　ターゲット

図形ウインドウポップアップメニューから起動するプラグインの例。

■simple_image_box : イメージボックスプラグイン　ターゲット

イメージボックスポップアップメニューから起動するプラグインの例。

■simple_importer : インポータプラグイン　ターゲット

イメージのエクスポータと形状のインポータの例。

■simple_modifier : モディファイアプラグイン　ターゲット

モディファイメニューから起動し、アンドゥをサポートするプラグインの例。

■simple_nib : Nibプラグイン　ターゲット（Macのみ）

Nibファイルから読み込んだダイアログボックスを表示する例。

■simple_plugin : シンプルプラグイン　ターゲット

単純で何の変哲もないプラグインの例。

■simple_renderer : レンダラプラグイン　ターゲット

レンダリング手法ポップアップメニューから選択できるカスタムレンダラの例。

Simple OpenGL Rendererは、OpenGLを使ってレンダリングする。

Simple Texture Rendererは、texture_interfaceを使って、Shade本体に表面材質の計算をさせる例。

■simple_resident : 常駐プラグイン　ターゲット

Shade起動時に読み込まれてそのまま常駐するプラグインの例。さまざまなコールバックに対応する方法も示す。

■simple_shader : シェーダプラグイン　ターゲット

マッピングポップアップメニューから選択できる、カスタムマテリアルシェーダの例。

■simple_window simple_window : ウインドウプラグイン　ターゲット

カスタムウインドウを定義するプラグインの例。

■simple_undo : アンドゥプラグイン　ターゲット

アンドゥ実装の例。
