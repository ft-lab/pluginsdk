説明

表面材質のマッピングの１つのレイヤを表すクラス。マッピングの情報の取得・設定に使用する。

以下の関数でこのクラスを直接取得する。これ以外に関数の引数として渡される場合がある。

surface_class::mapping_layer() , surface_interface::mapping_layer()
