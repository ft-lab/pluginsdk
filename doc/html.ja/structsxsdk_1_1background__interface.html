説明

背景の情報の取得、設定をする。

背景マッピングレイヤーの情報の取得、設定をするにはbackground_layer() によりbackground_layer_class の参照を取得する。

以下の関数でこのクラスを直接取得する。これ以外に関数の引数として渡される場合がある。

scene_interface::get_background_interface()
