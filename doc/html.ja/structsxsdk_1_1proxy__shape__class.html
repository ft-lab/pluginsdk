説明

プロクシ形状に関する操作を扱うクラス

リンク形状を含むシーンでは、レンダリングの対象としてのオブジェクトを通常の形状ツリーのオブジェクトからは特定できないため、レンダリングやエクスポート中にはリンクを展開したプロクシ形状のツリーが生成される。

プロクシ形状の親、子、姉、弟は（nilでなければ）かならずプロクシ形状

以下の関数でこのクラスを直接取得する。これ以外に関数の引数として渡される場合がある。

rendeing_context_interface::get_current_proxy_shape(), rendeing_context_interface::get_root_proxy_shape()
