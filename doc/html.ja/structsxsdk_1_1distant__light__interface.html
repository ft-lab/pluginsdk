説明

無限遠光源の情報の取得、設定に使用する。

無限遠光源の光源ごとの情報の取得、設定をするにはdistant_light_item() によりdistant_light_item_class の参照を取得する。

以下の関数でこのクラスを直接取得する。これ以外に関数の引数として渡される場合がある。

scene_interface::get_distant_light_interface()
