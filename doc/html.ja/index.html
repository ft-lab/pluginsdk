■はじめに

これは、Shade Plugin SDKに関するドキュメントであり、プラグイン開発に関する基礎知識やクラスリファレンス、スキーマ定義などについて記載しています。また、プラグイン開発には、C++および三次元グラフィックプログラミングに関する知識も必要とします。これらについては、詳しくは専門の書籍等を参考にしてください。
