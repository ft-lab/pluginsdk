void *  aux = 0 
) const [virtual]

イメージの各チャンネルの名前を取得する。1バイト文字のみで構成される。Shadeの表示言語に関わらず同じ名称となる。

マルチレイヤー対応の画像形式で保存する場合に、使用しないチャンネルについては、0が返される。

OpenEXRなど、マルチレイヤー対応形式で保存する場合のチャンネル名としても使用される。

引数:
indexチャンネルの番号。カラーの場合はそれぞれ、0=R, 1=G, 2=B, 3=Aのチャンネルとなる。グレイスケールの場合は0のみが有効。

virtual sxsdk::rgba_class sxsdk::image_layer_interface::get_maximum_value ( void *  = nullptr) const [virtual]

イメージで使用する明度の最大値(RGBA)を取得する。

HDRIなど、ピクセルの明るさが0.0 ～ 1.0の範囲に収まらない場合に使用する。イメージウインドウでの描画や画像ファイルへの保存時に、各ピクセルのRGBAの各要素はこの値を最大値として正規化される。

virtual void sxsdk::image_layer_interface::set_channel_name ( int  index,
const char *  name,
void *  aux = 0 
) [virtual]

イメージの各チャンネルの名前を設定する。デフォルトではRGBAが割り当てられるが、XYZ、UVなど、特別な名前を設定する場合に使用する。

マルチレイヤー対応の画像形式で保存する場合に、使用しないチャンネルについては、name に0を渡すことで保存対象からはずすことができる。0番目のレイヤー以外に0を渡すことでグレイスケールとして扱われる。

OpenEXRなど、マルチレイヤー対応形式で画像を保存する場合のチャンネル名としても使用される。

引数:
indexチャンネルの番号。カラーの場合はそれぞれ、0=R, 1=G, 2=B, 3=Aのチャンネルとなる。グレイスケールの場合は0のみが有効。
nameチャンネル名。1バイト文字のみの文字列を渡す。0を渡すと、未使用のチャンネルとして扱われる。
