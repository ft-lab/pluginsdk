説明

レンダリング時に使用される光源の情報を取得する。

rendering_context_interface は、無限遠光源、点光源、スポットライト、配光光源など全ての光源の情報を一括してテーブルに保持している。

レンダラプラグインからは、レンダリング中に変更される可能性があるシーン内の光源データではなく、rendering_light_class で光源データにアクセスするのが定石。

以下の関数でこのクラスを直接取得する。これ以外に関数の引数として渡される場合がある。

rendering_context_interface::get_light()
