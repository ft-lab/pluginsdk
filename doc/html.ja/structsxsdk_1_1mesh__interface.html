説明

コントロールポイントメッシュを扱うクラス

exporter_interface や renderer_interface でコールバック関数の引数として渡される。

Shadeの自由曲面は、一般的なベジェ曲面を独自拡張しており、互換性を持っていない。このクラスでは、自由曲面、球、掃引体、回転体から変換されたベジェ曲面を扱う。掃引体については側面だけがこのクラスで渡される。

以下の関数でこのクラスを直接取得する。これ以外に関数の引数として渡される場合がある。

part_class::create_control_mesh_interface()
