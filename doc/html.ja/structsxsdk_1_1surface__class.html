説明

表面材質の情報の設定、取得に使用する。

surface_interfaceとは異なり、インターフェイスオブジェクトを生成／削除するオーバヘッドなしに、表面材質属性にアクセスが可能。

surface_interfaceのget_xxx()関数は、相続元の設定値を（探して）取得するが、surface_classのget_xxx関数は、属性値が親から相続するように指定されているかどうかにかかわらず、そのsurface_classのメンバ変数の値をそのまま返す。

surface_class::find_xxx()関数は、相続元の設定値を探してそれを取得する。

以下の関数でこのクラスを直接取得する。これ以外に関数の引数として渡される場合がある。
