説明

ポリゴンメッシュ形状の一つの稜線を表すクラス。稜線の操作に使用する。

以下の関数でこのクラスを直接取得する。これ以外に関数の引数として渡される場合がある。

polygon_mesh_class::edge()

virtual void sxsdk::edge_class::remove ( void *  aux = 0) [virtual]

ポリゴンメッシュ形状から稜線を削除する。

ポリゴンメッシュの面と稜線は別構造になっているので、稜線を削除しても面にはまったく影響しないことに注意。
