説明

クリエータプラグインの基本クラス

creator_interfaceから派生させてカスタム属性クラスを定義することによって、クリエータプラグインを実装する。

クリエータプラグインは形状作成メニュー、パートメニューから起動され、図形ウインドウ内でのクリック／ドラッグ操作を待ち受ける。

オーバライド可。図形ウィンドウ内で円のドラッグ操作を受け付けるかどうかを返す。

accepts_sphere(), accepts_disk(), accepts_rectangle(), accepts_translation() は同時に使用することはできない。この関数がtrueを返す場合は、他の関数でfalseを返さなければならない。

ドラッグ操作の結果はset_disk()で取得する。

virtual void sxsdk::creator_interface::make_wireframe ( sxsdk::shape_class &  shape,
const sxsdk::mat4 &  mat,
int  view,
int  flags,
void *  aux = 0 
) [inline, virtual]

オーバライド可。Shadeが形状のワイヤフレームを表示する際にこの関数が呼ばれる。この関数をオーバーライドしてワイヤフレーム表示を追加することが可能。(参照：メモリ常駐型プラグイン)

sxsdk::shape_class::create_attribute_stream_interface_with_uuid() で引数wireframe_id にプラグインIDを渡して属性ストリームを生成すると、その形状に対して呼び出される。

この関数内でOpenGL命令を直接記述することで独自にワイヤーフレームを構築することができる。全体の処理速度に影響するため、この関数内ではワイヤフレームの描画以外の処理を行わないことを推奨。また、シーンの状態を変更する処理の動作は保証されない。

引数:
shape対象となっている形状
mat形状のローカル座標系からワールド座標系に変換するマトリクス
view描画対象のビュー（0:側面 1:正面 2:上面 3:透視)
flagsバウンディングモード、ピックモードを判別するフラグ
