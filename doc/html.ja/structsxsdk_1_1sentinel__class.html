説明

パートの終端を表す番兵形状を扱うクラス

番兵形状はパートの終端を表し、すべてのパートに必ず存在する(空のパートであっても同様)。

番兵形状のbroはパートの一番上の形状、sisはパートの一番下の形状になる。

has_bro(), has_sis(), has_son() で、確認する形状が番兵だった場合はfalseを返す。

sentinel_classは内部的に使用するクラスのため、プラグイン側からの取得をしない。
