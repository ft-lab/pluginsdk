説明

カスタム属性プラグインの基本クラス。

attribute_interface から派生させてカスタム属性クラスを定義することで、カスタム属性プラグインを実装する。

カスタム属性プラグインは、accepts系関数をオーバーライドすることで、カスタム情報メニューのほか、ブラウザ・カメラ・無限遠光源・表面材質・表面材質のマッピングの情報ポップアップメニューから起動される。

形状・カメラ・無限遠光源・表面材質・表面材質のマッピングにプラグイン独自のカスタム属性情報を付加する機能を実装するために使用する。

カスタム属性情報の付与にはstream_interface を使用する。付与したカスタム情報はそのプラグインまたは別のプラグインからその情報にアクセスすることができる。

また、形状に属性を付与することでray_intersection() による交点計算のカスタムなどを行うことができる。

オーバライド可。交点計算に必要な独自の情報を生成する。

custom_element_info_base_classの派生クラスを作成し、そのポインタを返す。

ここで生成したオプジェクトのポインタは、get_bounding_box() およびray_intersection() 関数呼び出しの引数として渡される。

stream_interfaceを介した情報の読み書きは動作が遅いため、ここで必要な情報をオブジェクトに読み込んでおき、交点計算で使用する。

get_bounding_box() およびray_intersection() 関数は複数スレッドから同時に呼び出される可能性があるので、shader_interface に情報を渡すなど、スレッドごとに扱う必要のある場合はnew_per_thread_info() で生成する。

レンダリング終了時に自動的にdeleteされる。

交点計算に独自の情報を確保しておく必要がなければ、new_custom_element_info() をオーバライドする必要はない。

作成したcustom_element_info_base_classの派生クラスを返す。
