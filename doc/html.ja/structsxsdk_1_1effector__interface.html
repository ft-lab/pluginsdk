説明

エフェクタプラグインの基本クラス

effector_interfaceから派生させたクラスを定義することによって、エフェクタプラグインを実装する。

典型的な使い方としては、レンダリング計算後にコールバック関数が呼ばれ、そのコールバック関数内でレンダリングイメージに対して操作することになる。

virtual bool sxsdk::effector_interface::can_ask ( void *  aux = 0) [inline, virtual]

オーバライド可。ask()関数呼び出しに対応するかどうかを返す。

この関数をオーバライドしてtrueを返すと、レンダリングオプション設定のSet...ボタンがアクティブになり、ユーザがそれをクリックすることによってask()関数が呼ばれる。ask()関数をオーバライドした場合にはcan_ask()でtrueを返す。
