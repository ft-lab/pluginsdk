説明

自由曲面、球、掃引体、回転体から変換されたポリゴンメッシュを扱うクラス

exporter_interface や renderer_interface でコールバック関数の引数として渡される。

完全な格子構造のメッシュのみを扱い、距離補正とパラメータ補正のUVを持っている。掃引体については側面だけがこのクラスで渡される。

以下の関数でこのクラスを直接取得する。これ以外に関数の引数として渡される場合がある。

sxsdk::shape_class::create_converted_meshes_interface()
