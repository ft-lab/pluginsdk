説明

レンダラプラグインの基本クラス

renderer_interfaceから派生させたクラスを定義することによって、レンダラプラグインを実装する。

レンダラプラグインは、レンダリング設定の手法ポップアップメニューに追加され、レンダリングの際に使用される。

exporter_interface から派生しているので、レンダリング時の形状取得にはエクスポータプラグインと同じコールバックを使用することができる。（コールバックの呼び出し開始には、render() 関数内でrendering_context_interface::convert() を呼び出す）

virtual bool sxsdk::renderer_interface::uses_maximum_distance ( void *  aux = 0) [inline, virtual]

オーバライド可。マルチパスレンダリングのZ値の「最大距離」の設定をレンダラで使うかどうかを返す。

レンダリング設定の表示を設定する。true を返す場合、rendering_interface::get_auto_maximum_distance() がtrueのときは、無限遠とする距離(カメラ平面から最も遠い点までの距離＋α)をプラグイン側で計測して、Z値、Z値(n次レイ)のレイヤーにimage_layer_interface::set_maximum_value() に設定しておく必要がある。

Z値、Z値(n次レイ)のパスでは、ピクセルにカメラ平面から形状までの到達距離を実寸(mm)で設定する。
