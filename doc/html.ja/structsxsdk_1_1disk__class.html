virtual sxsdk::mat4 sxsdk::disk_class::get_matrix ( void *  = nullptr) const [virtual]

円を定義する変換マトリクスを取得する。

単位円（原点に位置し半径が1、X-Y平面上にある円）から対象とする円に変換する4x4座標変換マトリクス。unmatrix関数でこのマトリクスをスケール、回転、移動の成分に分解することができ、この場合、それぞれの成分は円の半径、向き、位置ということになる。

円を定義する変換マトリクスを設定する。

単位円（原点に位置し半径が1、X-Y平面上にある円）から対象とする円に変換する4x4座標変換マトリクス。unmatrix関数でこのマトリクスをスケール、回転、移動の成分に分解することができ、この場合、それぞれの成分は円の半径、向き、位置ということになる。
