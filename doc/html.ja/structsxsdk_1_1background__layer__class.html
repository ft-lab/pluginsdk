説明

一個の背景レイヤを表すクラス。背景レイヤの情報の取得、設定に使用する。

背景の設定は０個以上のレイヤから構成され、background_layerはその一個のレイヤを表す。

以下の関数でこのクラスを直接取得する。これ以外に関数の引数として渡される場合がある。

background_interface::background_layer()
