Detailed Description

importer plugin base class

Implement the importer plugin by defining an importer_interface derived class.

The importer plugin is run from the Import menu.

virtual bool sxsdk::importer_interface::can_import_from_stream ( void *  aux = 0) [inline, virtual]

Overridable. Returns whether to import from the stream.

If returns false, Shade calls do_import() with the import stream interface as a null pointer, without displaying the Open dialog.

Therefore, the importer plugin can select and load a file on its own.
