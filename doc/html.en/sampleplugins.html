Sample Plugin Project

*multi_plugin : Multiple Plugin Target

This sample code shows you how to implement two plugin components in a single plugin entity, i.e. a dll file for Windows and a bundle for Mac OS X.

*simple_attribute : Simple Attribute Plugin Target

This sample code shows you how to attach custom persistent attribute to various types of objects and to read, write and use it.

*simple_browser : Simple Browser Plugin Target

This sample code shows you how to write a plugin that is invoked from the browser popup menu.

*simple_creator : Simple Creator Plugin Target

This sample code shows you how to write a plugin that is invoked from the create menu and responds to user's clicking and dragging in the figure window.

*simple_dialog : Simple Dialog Plugin Target

This sample code shows you how to create and display a dialong box on the fly.

simple_effector : Simple Effector Plugin Target

This sample code shows you how to apply custom effects to the rendered image.

*simple_exporter : Simple Exporter Plugin Target

This sample code shows you how to write an image exporter and shape object exporter.

*simple_figure : Simple Figure Plugin Target

This sample code shows you how to write a plugin that is invoked from the figure window popup menu.

*simple_image_box : Simple Image Box Plugin Target

This sample code shows you how to write a plugin that is invoked from an image box popup menu.

*simple_importer : Simple Importer Plugin Target

This sample code shows you how to write an image importer and shape object importer.

*simple_modifier : Simple Modifyer Plugin Target

This sample code shows you how to write a plugin that is invoked from the modify menu. It also shows you how to implement simple undo operations.

*simple_nib : Simple Nib Plugin Target (Mac only)

This sample code shows you how to display a dialog box from a nib file

*simple_plugin : Simple Plugin Target

This sample code shows you how to write a simple and generic plugin.

*simple_renderer : Simple Renderer Plugin Target

This sample code shows you how to write a custom renderer plugin that can be selected from the rendering methods popup menu.

Simple OpenGL Renderer is a simple example for using OpenGL.

Simple Texture Renderer shows you how to use texture_interface to delegate surface attribute calculation to Shade.

*simple_resident : Simple Resident Plugin Target

This sample code shows you how to write a resident plugin which gets loaded when Shade starts up and will not be unloaded until Shade terminates. It also shows you how to respond to various callbacks.

*simple_shader : Simple Shader Target

This sample code shows you how to write custom material shaders that can be selected from mappings popup menu.

*simple_window simple_window : Simple Window Target

This sample code shows you how to create custom windows.

*simple_undo : Simple Undo Target

This sample code shows you how to implement undo operations.
