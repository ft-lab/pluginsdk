Detailed Description

Interface for accessing custom joints.

Custom joints only have a value, and unlike other joints to not affect child objects. Used as an interface to change the plug-in settings or to create independent joints with a callback function.

Custom joint settings are stored as part of the object attributes, but to make it easy to access from plug-ins and scripts the settings can be get/set from a separate interface.
