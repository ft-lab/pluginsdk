virtual sxsdk::mat4 sxsdk::disk_class::get_matrix ( void *  = nullptr) const [virtual]

Get the transformation matrix that defines the disk.

A 4x4 transformation matrix that transforms a unit disk (a disk with 1 and lies on the X-Y plane, positioned at the origin) to the target disk. You can decompose this matrix into scale, rotation and translation components using the unmatrix() function. Each component thus corresponds to radius, orientation and position of the target disk.
