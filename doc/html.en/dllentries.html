Shared Global Functions

*Overview

Shade plugins must be opened outside specific global functions called from Shade. These functions are basic functions necessary for Shade to access plugins, and are called either when Shade starts or when a plugin is run from Shade. Details and the prototype are listed in dllentries.h.
