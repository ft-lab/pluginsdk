Detailed Description

The effector base class.

Implement effector plug-ins by defining effector_interface derived classes.

Typically, a callback function is called after the rendering calculation, and the operation is done on the renderingn image within that callback function.

virtual bool sxsdk::effector_interface::can_ask ( void *  aux = 0) [inline, virtual]

Overridable. Returns whether or not the effector responds to /a ask() function call.

If you override this function and return true, the Set... button of the rendering settings palette becomes active and a user can click it to call the ask() function. When you override the ask() function, you should override this function and return true.
