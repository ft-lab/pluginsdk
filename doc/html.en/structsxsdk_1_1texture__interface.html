Detailed Description

Plugins can use Shade's internal shading calculation routines via texture_interface.

texture_interface maintains its internal states and does shading calculation using those states.

A typical usege from a plugin is to create a texture_interface object, then to repeat setting up the internal states, calling update() then fetching the results.
