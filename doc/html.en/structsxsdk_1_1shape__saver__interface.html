Detailed Description

Retains geometric data of objects for undo support.

An undo task unit (executed when selected by the user from the Undo menu) comprises one or more registered undo actions.

Geometric state of the original shape object is saved to a shape_saver_interface object when it is created.

A plugin must register actions that revert the geometric state of objects to the previous one (and back to the current state) as an undo action. After the geometry state is saved to the shape_saver_interface object, you modify the original shape object, then you must call set_undo_action() function to register the undo action.

See the simple_undo target of the SDK sample project for how to use shape_saver_interface.
