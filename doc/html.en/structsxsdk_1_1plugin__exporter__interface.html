Detailed Description

Saves all types of export info (only valid during an export).

Used when getting scene info or streams for export operations in exporter_interface derived exporter plugins.

Passed as an argument of exporter_interface::do_export().

The following function gets this class directly. It may also be passed as a function argument.

rendering_context_interface::get_plugin_exporter_interface()
