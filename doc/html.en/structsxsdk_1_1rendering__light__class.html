Detailed Description

Gets info for lights used during rendering.

The rendering context contains all the information regarding light sources including distance light, point light, spot light and distribution light in a table.

Plugin should use those information rather than accessing light source information in the scene which may be changed while rendering.

The following function gets this class directly. It may also be passed as a function argument.

rendering_context_interface::get_light()
