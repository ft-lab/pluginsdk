Detailed Description

Creator plug-in base class.

By deriving creator_interface and defining a custom attribute class, you can implement a creator plug-in.

Creator plug-ins are started from the Create or Part menus and wiat for a click or drag action in the Figure window.

virtual void sxsdk::creator_interface::make_wireframe ( sxsdk::shape_class &  shape,
const sxsdk::mat4 &  mat,
int  view,
int  flags,
void *  aux = 0 
) [inline, virtual]

Overridable. This function is called when Shade displays the shape's wireframe. The default behavior is to do nothing, but you can override this function and add your own wireframe display.

If the plugin ID is passed to the argument wireframe_id with sxsdk::shape_class::create_attribute_stream_interface_with_uuid() and generates an attribute stream, this function is called for that object.

By making a direct OpenGL command within this function, you can construct an independent wireframe. Because this affects the whole processing speed, only drawing wireframes within this function is recommended. Also, processes that modify the scene are not guaranteed.

Parameters:
shapethe target object
matthe local to world coordinate matrix
viewthe target drawing view (0: Side 1: Front 2: Top 3: Pers)
flagsthe flag that determines Bounding mode or Pick mode
