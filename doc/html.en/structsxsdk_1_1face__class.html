Detailed Description

Represents a face of a polygon mesh object.

The following function gets this class directly. It may also be passed as a function argument.

Get the list of distance-UV values.

The list of distance-UV values assigned to each vertex of the face. If the polygon mesh object has been created by converting from a curved surface object, these distance-UV values are initialized with the normalized geometric distance values along u and v directions of the original curved surface.

virtual int sxsdk::face_class::get_normals ( sxsdk::vec3  list[],
void *  = nullptr 
) const [virtual]

Get normal vectors.

Normal vectors for polygon mesh faces are usually calculated on the fly when needed such as rendering. If you explicitly set the normal vectors for a face, those normal vectors are used instead of calculating ones. The number of items must be same as the number of vertices of the face. Or, the normal vectors are cleared if the number of items is zero.
