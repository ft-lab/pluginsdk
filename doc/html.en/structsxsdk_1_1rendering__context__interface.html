Detailed Description

Contains rendering related infomation that are mostly valid only while rendering.

The following function gets this class directly. It may also be passed as a function argument.

virtual sxsdk::mat4 sxsdk::rendering_context_interface::get_world_to_device_matrix ( void *  = nullptr) const [virtual]

Get The matrix to transform from the world coordinates to the device coodinates.

The matrix that moves and rotates the world coordinates based on the camera position, direction, and bank. The origin is equal to the eye position, and the eye direction is the Z axis.

Device coordinates, here, means the coordinates of the rendering image (with the origin at the upper left corner and in pixel unit).

virtual sxsdk::mat4 sxsdk::rendering_context_interface::get_world_to_perspective_matrix ( void *  = nullptr) const [virtual]

Get The matrix to transform from the world coordinates to the perspective coodinates.

The matrix that moves and rotates the world coordinates based on the camera position, direction, and bank. The origin is equal to the eye position, and the eye direction is the Z axis.

Perspective coordinates, here, means the coordinates that are normalized to [-1, 1] in height and width dimension and [0, 1] in depth dimension. Outside this range clipping occurs in this coordinate system.

virtual sxsdk::mat4 sxsdk::rendering_context_interface::get_world_to_view_matrix ( void *  = nullptr) const [virtual]

Get The matrix to transform from the world coordinates to the view coodinates.

The matrix that moves and rotates the world coordinates based on the camera position, direction, and bank. The origin is equal to the eye position, and the eye direction is the Z axis.

The Perspective conversion is irrelevant.
