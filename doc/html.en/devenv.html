Development Environments

*Plugin Development Environments

The development language of Shade Plugin SDK is C++. One of the development environments listed below is required by Shade Plugin SDK.

Windows

Operating System

  • Windows 7 32bit / 64bit
  • Windows Vista 32bit / 64bit
  • Windows XP Professional x64 Edition
  • Windows XP Professional / Home Edition / Media Center Edition (Service Pack 2 or above)

Development Tools

  • Visual Studio 2010 Ultimate / Premium / Professional Edition
  • Visual C++ 2010 Express Edition

※ Windows 98 / Me / NT / 2000 are not supported.
※ Visual C++ 6.0 is not supported.
※ Visual Studio .NET Version 2002, 2003 are not supported.
※ Visual C++ 2010 Express Edition is not supported 64-bit development of Shade Plugin.
※ Service Pack 1 is highly recommend when using Visual Studio 2010.

　 Visual Studio 2010 Service Pack 1
　 http://www.microsoft.com/en-us/download/details.aspx?id=23691

※ To use Visual C++ 2010 Express Edition for plugin development. Please reference to *Use Visual C++ 2010 Express Edition for Plugin Development

OR

  • Visual Studio 2008 Team System / Professional Edition / Standard Edition
  • Visual C++ 2008 Express Edition

※ Windows 98 / Me / NT / 2000 are not supported.
※ Visual C++ 6.0 is not supported.
※ Visual Studio .NET Version 2002, 2003 are not supported.
※ Visual C++ 2008 Express Edition is not supported 64-bit development of Shade Plugin.
※ Service Pack 1 is highly recommend when using Visual Studio 2008.

　 Visual Studio 2008 Service Pack 1
　 http://www.microsoft.com/en-us/download/details.aspx?id=10986

※ To use Visual C++ 2008 Express Edition for plugin development. PLease reference to *Use Visual C++ 2008 Express Edition for Plugin Development

Mac OS X

Operating System

  • Mac OS X 10.6 or later

Development Tools

  • Xcode 3.2 later / Xcode 4.3 / Xcode 4.4

※ Classic / Mac OS 9 /Mac OS X 10.5 and previous versions are not supported.
※ CodeWarrior, Project Builder, Xcode3.2 are not supported.
※ To use Xcode 4.4, project settings need to modify.

*Plugin Development with Xcode 4.4

You can use Xcode 4.4 under Mac OS X 10.8 (Mountain Lion) for Shade Plugin development, however, you need to change some settings in your plugin project file as shown at the followings.

In your plugins project, go to "Build Settings" and under the "Architectures" group, change the Base SDK to "OS X 10.7" instead of "OS X 10.6".
Then your plugin will run under both OS X 10.7 and 10.8.
However, if you want to run your plugin under 10.6, then you should use Xcode 4.3.

*Supported Shade Level

Plugins created with Shade 13 Plugin SDK works in Shade 13 Professional and Standard version. Basic version is not supported plugins.
