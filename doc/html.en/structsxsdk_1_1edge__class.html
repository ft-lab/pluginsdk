Detailed Description

Represents an edge of a polygon mesh object.

The following function gets this class directly. It may also be passed as a function arugment.

polygon_mesh_class::edge()

Member Function Documentation

virtual void sxsdk::edge_class::remove ( void *  aux = 0) [virtual]

Removes the edge from the polygon mesh object.

Note that since faces and edges of a polygon mesh objects are independently maintained, removing an edge does not affect faces at all.
