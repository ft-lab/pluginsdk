Detailed Description

Control point mesh class.

Passed as a callback argument with exporter_interface and renderer interface.

Shade curved surfaces are not compatible with general Bezier curves. This class handles Bezier curves converted from curved surfaces, spheres, extruded solids, and revolved solids. Only the sides of extruded solids are passed with this class.

The following function gets this class directly. It may also be passed as a function argument.

part_class::create_control_mesh_interface()
