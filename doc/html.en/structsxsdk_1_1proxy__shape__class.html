Detailed Description

proxy shape

Since an object being rendered cannot always be specified from those in the regular shape object tree in a scene, special tree with proxy shapes is created while rendering.

Proxy shape's dad, son, sis, bro are always proxy shapes (if not nil).

The following functions get this class directly. It may also be passed as a function argument.

rendeing_context_interface::get_current_proxy_shape(), rendering_context_interface::get_root_proxy_shape()
