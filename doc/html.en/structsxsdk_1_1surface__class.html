Detailed Description

Used to set, get surface attributes.

Allows you to access surface attributes without overhead of creating/deleting interface objects.

surface_class::get_xxx() functions returns surface_class's member variable whether the attribute is set to inherit from an ancestor or not while surface_interface::get_xxx() functions searches for an ancestor and returns its value.

surface_class::find_xxx() functions searches for an ancestor and returns its value.

The following function gets this class directly. It may also be passed as a function argument.

sxsdk::shape_class::get_surface()
