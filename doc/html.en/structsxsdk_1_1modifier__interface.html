Detailed Description

Modifier plugin base class.

Implement the modifer plugin by defining a modifier_interface derived class.

The modifier plugin is run from the Modify menu, and waits for a click or drag in the Figure window.

virtual void sxsdk::modifier_interface::make_wireframe ( sxsdk::shape_class &  shape,
const sxsdk::mat4 &  mat,
int  view,
int  flags,
void *  aux = 0 
) [inline, virtual]

Overridable. This function is called then Shade displays the shape's wireframe. The default behavior is to do nothing, but you can override this function and add your own wireframe display.

If you pass the plugin ID to the argument wireframe_id with sxsdk::shape_class::create_attribute_stream_interface_with_uuid() amd generate the attribute stream, this function is called for that object.

An independent wireframe can be constructed by giving a direct OpenGL command within this function. Because this affects the overall processing speed, it is recommended to only do wireframe drawing in this function. Also, scene modifications are not guaranteed.

Parameters:
shapethe target object for drawing
matthe local to world coordinates matrix
viewthe drawing view (0: Side 1: Front 2: Top 3: Pers)
flagsbounding mode or pick mode
