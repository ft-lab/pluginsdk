Detailed Description

Represents one image. Used to access images.

Shade 7 and earlier had both 32 bit (8 bits per channel) and 128 bit (32 bits per channel) real color images, but Shade 8 has combined these into one image format.

Image color depths of 32, 64, and 128 bits are supported. (Set in the Rendering Settings for rendered images, and by the file format for image files.)

As such, the plugin can set/get image info without worrying about the image color depth.

64 and 128 bit images support HDRI and can hold a brightness value greater than 1.0. Because converting the color value from the image to an integer can cause overflow and precision errors, floating point values are recommended for most cases.

sxsdk::rgb_class and rgba_class which use floating point values for the color are used in the following functions. Because 32 bit images can also be used it is not necessary to switch the function according to the color depth.

set_real_color(), get_real_color(), set_real_colors(), get_real_colors(), set_pixel_rgba_float(), get_pixel_rgba_float(), set_pixels_rgba_float(), get_pixels_rgba_float()

The following functions get this class directly. It may also be passed as a function argument.

shade_interface::create_image_interface(), shade_interface::create_image_interface_from_file()
