Detailed Description

Class that handles sentinel objects, which represent the terminus of parts.

The sentinel object represents the terminus of a part, and exists for all parts (even empty parts).

The sentinel object's bro is the topmost object in the part, and the sis is the bottom-most object in the part.

Returns false with has_bro(), has_sis(), and has_son() if the object to confirm is a sentinel.

Because sentinel_class is used internally, it is not returned from the plug-in side.
