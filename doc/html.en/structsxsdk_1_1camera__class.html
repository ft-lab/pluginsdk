Detailed Description

Represents a camera.

Multiple cameras can be definied in the scene; however camera_class represents one camera.

The following function gets this class directly. This class my also be passed as an argument by another function.

scene_interface::get_camera()

virtual sxsdk::mat4 sxsdk::camera_class::get_world_to_perspective_matrix ( void *  = nullptr) const [virtual]

Get the matrix that transforms the the view coordinate system of the camera to the world coordinate system.

The matrix that moves, rotates, and normalizes the world coordinates based on the camera position, direction, and bank. The origin is equal to the eye position, and the eye direction is the Z axis.

Normalizes the coordinates so that they fit in the field of view with -1.0 to 1.0 vertically and horizontally, and going from 0.0 to 1.0 deep.

virtual sxsdk::mat4 sxsdk::camera_class::get_world_to_view_matrix ( void *  = nullptr) const [virtual]

Get the matrix that transforms the world coordinate system to the view coordinate system of the camera.

The matrix that moves and rotates the world coordinates based on the camera position, direction, and bank. The origin is equal to the eye position, and the eye direction is the Z axis.

Perspective conversions are irrelevant.
