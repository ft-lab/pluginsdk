Plugin IDs

*UUID

Shade 8 uses 128 bit UUIDs instead of 32 bit integer IDs to idenfy pugins. No need to register and manage ID ranges for developers to avoid collision. Each plugin developer can just use uuidgen command to generate one when needed.

Legacy plugins that use 32 bit integer IDs should run normally without change because Shade 8 internally expands 32 bit ID to 128 bit UUID.

*How to generate UUID

Windows XP / Vista / 7 + Visual C++ 2008/2010

  • Create GUID from the Visual Studio 2010 Tools menu.
  • Copy the generated UUID to the clipboard with the Copy button.
  • Edit the copied UUID in the following format.
 // {8103127E-6E91-467f-8B98-9E6821DCC7CE}
 const uuid_class SAMPLE_UUID = uuid_class("8103127E-6E91-467f-8B98-9E6821DCC7CE");

Mac OS X

  • Execute uuidgen command in a terminal.
  • Edit the generated UUID in the following format.
 const uuid_class SAMPLE_UUID = uuid_class("8103127E-6E91-467f-8B98-9E6821DCC7CE");
