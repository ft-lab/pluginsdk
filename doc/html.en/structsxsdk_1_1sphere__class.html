virtual sxsdk::mat4 sxsdk::sphere_class::get_matrix ( void *  = nullptr) const [virtual]

Get the transformation matrixthat defines the sphere.

A 4x4 transformation matrix that transforms a unit sphere (a shere with 1 and aligned to Y axis, positioned at the origin) to the target sphere. You can decompose this matrix into scale, rotation and translation components using the unmatrix() function. Each component thus corresponds to radius, orientation and position of the target sphere.
