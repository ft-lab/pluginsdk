Detailed Description

Interface for accessing slider joints.

Slider joint settings are stored as part of the object attributes, but to make it easy to access from plug-ins and scripts the settings can be get/set from a separate interface.

The following function gets this class directly. It may also be passed as a function argument.

sxsdk::shape_class::get_slider_joint_interface()
